# Possible Simulations

There are common simulation targets.

 * Manufacturing - flow of materials and products like the job-shop scheduling problem (JSSP).

 * Logistics and transportation - traffic flow, congestion.

 * Healthcare systems - ERs, ambulances.

 * Computer networks.

 * Inventory management - classic.

 * Financial modeling - transaction processing, risk management.

 * Flight simulators, autopilot.

 * Game engines.
